# 咕咕機器人 v2.0
## 使用
```
'''
咕咕機器人 (實做)

請在 config.py 設定好 token 再啟動，
並確保已經在 @BotFather 關閉了 Privary 設定：

/setprivary  -> 設定 Disable

接著私訊您的機器人，或者是放到您的群組即可。
'''
```

## 函式庫
libs.py 包含 GET/POST TG 機器人的函式庫與
隨機長度字元產生器。

此處不提供說明，請自行參閱 libs.py 內容

此外， `useless_data.py` 包含廢棄無用的函式庫，
但不保證是否可以執行。
